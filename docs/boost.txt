boost on windows
================

- download source archive and unpack it
- open the visual studio command line tool
- cd-ing into unpacked boost folder
- run `bootstrap.bat` script
- run `b2 toolset=msvc-10.0 --with-filesystem --with-system --with-date_time --with-regex` to build boost for Visual Studio 2010
- look at `stage` subfolder content


dependencies
============

The follosing libraries are used by C++ framework:
- boost.system

The following libraries are used for testing:
- boost.filesystem
